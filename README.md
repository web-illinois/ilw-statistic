# ilw-statistic
Statistic Component for the Illinois Web Toolkit

## Overview

Hero component for the web toolkit. The Hero Image component adds a large picture at the top of the page. It is used to draw attention to the page and can be used to set the tone for the page.

```
Overall ranking among undergraduate programs in the U.S.
  <span slot="source">U.S. News and World 2024</span>
</ilw-statistic>
```



## Accessibility Notes and Use

The use of HTML statistics with text in place of graphical statistics can help make your content accessible without the need for additional steps of integrating alt text.
